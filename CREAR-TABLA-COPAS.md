# Crear Tabla de Copas en Supabase

Para que el botón "Crear Copa" funcione y los datos se guarden permanentemente sin afectar los partidos existentes, necesitamos crear una pequeña tabla nueva.

1. Abre **Supabase** > **SQL Editor** (icono de hoja con código en la barra lateral izquierda)
2. Haz clic en **"New Query"**
3. Pega este código:

```sql
create table cups (
  id bigint generated by default as identity primary key,
  name text not null,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Habilitar acceso público (igual que la tabla matches)
alter table cups enable row level security;

create policy "Enable read access for all users" on cups for select using (true);
create policy "Enable insert for all users" on cups for insert with check (true);
create policy "Enable delete for all users" on cups for delete using (true);
```

4. Haz clic en **"Run"** (botón verde abajo a la derecha)
5. ¡Listo! Ahora la app podrá guardar copas.
