# Torneo Nacional EFIL 2025

Aplicación web para gestionar el Torneo Nacional EFIL 2025.

## Características

- **Vista Pública**: Tabla de posiciones, resultados y fixture
- **Panel Admin**: Gestión de grupos, equipos, partidos y alertas
- **Credenciales Admin**: 
  - Usuario: `admin`
  - Contraseña: `efil2025`

## Tecnologías

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- LocalStorage para persistencia de datos

## Uso Local

Simplemente abre `index.html` en tu navegador.

## Deploy en GitHub Pages

Esta aplicación está configurada para funcionar en GitHub Pages.
